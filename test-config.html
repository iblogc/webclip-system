🔧 Web收藏脚本测试页面

用于测试油猴脚本的配置功能

📋 测试步骤

1. 安装脚本

确保已在Tampermonkey中安装了最新版本的Web收藏脚本

2. 首次配置

如果是首次使用，应该看到右下角的⚙️配置按钮

3. 配置测试

点击配置按钮，测试以下功能：

  • 填写Gist ID和GitHub Token
  • 修改分类选项
  • 调整按钮位置和大小
  • 更改按钮颜色
  • 切换各种开关选项

4. 收藏测试

配置完成后，按钮应变成📌，测试收藏功能

🎛️ 配置功能清单

  • ✅ 基础配置: Gist ID, GitHub Token
  • ✅ 分类管理: 自定义分类选项
  • ✅ 按钮位置: 四个角落可选
  • ✅ 按钮大小: 40-80px滑块调节
  • ✅ 按钮颜色: 颜色选择器
  • ✅ 自动聚焦: 开关控制
  • ✅ 通知设置: 开关和持续时间
  • ✅ 配置重置: 一键恢复默认

🎯 交互测试

左键点击

配置完成后，左键点击📌按钮应该打开收藏弹窗

右键点击

右键点击📌按钮应该打开配置界面

键盘快捷键

在弹窗中测试以下快捷键：

  • Esc - 关闭弹窗
  • Ctrl/Cmd + Enter - 提交表单

⚠️ 注意事项

  • 首次使用需要配置Gist ID和GitHub Token
  • 配置数据保存在浏览器本地存储中
  • 右键菜单可能被浏览器默认菜单覆盖，请快速点击
  • 如果按钮不显示，请检查脚本是否正确安装和启用
